# Exo 00 - Installation et premier dossier web
Preparer un environnement de travail simple pour tout le cours.
A la fin, tu dois avoir un dossier propre contenant un fichier HTML ouvert dans ton navigateur.


## Consigne

Cree un dossier appele `cours-html-css`.

Dans ce dossier, cree :

```text
cours-html-css/
  exo-01/
      index.html
      style.css
      images/
```

Pour l'instant, le fichier `style.css` et le dossier `images` peuvent rester vides.

### Etape 4 - Ajouter un premier contenu

Dans `index.html`, ecris :

```html
Bonjour le web !
```

### Etape 5 - Ouvrir dans le navigateur

Double-clique sur `index.html` ou fais clic droit puis "ouvrir avec" ton navigateur.

## Resultat attendu

Le navigateur affiche :

```text
Bonjour le web !
```

## Checklist avant de rendre

- [ ] Le dossier s'appelle `cours-html-css`.
- [ ] Le fichier s'appelle `index.html`.
- [ ] Le fichier s'ouvre dans un navigateur.
- [ ] Le texte apparait bien.

## Bonus

Change le texte en :

```text
Bienvenue dans mon premier site web.
```

Puis recharge la page dans le navigateur.
