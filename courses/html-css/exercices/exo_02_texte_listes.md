# Exo 02 - Organiser du texte et creer des listes

## Moment strategique

A faire apres les slides sur les titres, paragraphes, mise en valeur et listes.

## Objectif

Creer une page lisible avec des titres, des paragraphes, du texte mis en valeur et plusieurs types de listes.

## Ce que tu vas apprendre

- Utiliser les titres `h1`, `h2`, `h3`.
- Ecrire des paragraphes avec `p`.
- Mettre du texte en valeur avec `strong`, `em`, `mark`.
- Creer une liste a puces avec `ul`.
- Creer une liste ordonnee avec `ol`.

## Consigne

Complete ta page pour presenter un sujet simple.

Exemples :

- mes 3 films preferes ;
- ma recette favorite ;
- mon programme de formation ;
- les services d'une petite entreprise fictive ;
- les activites d'un club.

## Structure attendue

Ta page doit contenir au minimum :

1. Un `h1` pour le titre principal.
2. Deux `h2` pour deux grandes parties.
3. Au moins deux paragraphes.
4. Une liste non ordonnee `ul`.
5. Une liste ordonnee `ol`.
6. Au moins un mot en `strong`.
7. Au moins un mot en `em`.
8. Au moins un mot en `mark`.

## Etapes detaillees

### Etape 1 - Choisir le sujet

Choisis un sujet facile a expliquer en quelques lignes.

Exemple :

```text
Sujet : ma recette de crepes
```

### Etape 2 - Ajouter un titre principal

```html
<h1>Ma recette de crepes</h1>
```

### Etape 3 - Ajouter une introduction

```html
<p>Cette recette est simple et rapide a preparer.</p>
```

### Etape 4 - Ajouter une liste a puces

Une liste a puces sert quand l'ordre n'est pas tres important.

Exemple :

```html
<h2>Ingredients</h2>
<ul>
    <li>Farine</li>
    <li>Oeufs</li>
    <li>Lait</li>
</ul>
```

### Etape 5 - Ajouter une liste ordonnee

Une liste ordonnee sert quand l'ordre est important.

Exemple :

```html
<h2>Etapes</h2>
<ol>
    <li>Melanger les ingredients.</li>
    <li>Faire chauffer la poele.</li>
    <li>Cuire les crepes.</li>
</ol>
```

### Etape 6 - Mettre certains mots en valeur

Exemples :

```html
<p>Il est <strong>important</strong> de bien melanger la pate.</p>
<p>Servez les crepes <em>chaudes</em>.</p>
<p><mark>Attention</mark> a ne pas trop cuire la pate.</p>
```

## Resultat attendu

La page doit etre plus organisee qu'un simple bloc de texte. On doit comprendre rapidement les parties de la page.

## Bonus

Ajoute une liste de definitions avec `dl`, `dt` et `dd` pour expliquer trois mots importants de ton sujet.
