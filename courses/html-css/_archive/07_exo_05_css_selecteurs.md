# Exercice 05 - Relier CSS et utiliser les premiers sélecteurs

> À faire dans le PDF : après la page/slide 56, quand la liaison CSS et les sélecteurs ont été vus.

## Objectif

Commencer la mise en forme avec un fichier CSS séparé.

## Énoncé

Ajoute un fichier `style.css` à ton exercice et relie-le correctement à ta page HTML. Applique ensuite plusieurs styles simples pour vérifier que tu sais cibler des éléments de différentes manières.

Ta page doit rester lisible : cherche une mise en forme claire, pas une décoration excessive.

## Contraintes

- Le CSS doit être écrit dans `style.css`.
- Le fichier CSS doit être lié depuis le `head` de la page HTML.
- Le style général de la page doit être modifié.
- Au moins un sélecteur de balise doit être utilisé.
- Au moins une classe réutilisable doit être créée et utilisée dans le HTML.
- Au moins un identifiant doit être ciblé en CSS.
- Au moins une règle doit grouper plusieurs sélecteurs.
- Le CSS ne doit pas être écrit directement dans les balises HTML.

## Résultat attendu

La page doit avoir une apparence différente de la version sans CSS : police, titres, couleurs ou blocs doivent montrer que le fichier CSS est bien actif.

## Checklist

- `style.css` existe.
- Le fichier CSS est relié dans le `head`.
- Le style général de la page change.
- Un sélecteur de balise est utilisé.
- Une classe est utilisée.
- Un identifiant est utilisé.
- Un groupe de sélecteurs est utilisé.
- Le HTML et le CSS restent séparés.

## Bonus

Transforme un lien en bouton visuel grâce à une classe CSS.
