# Exercice 06 - Typographie, couleurs et arrière-plans

> À faire dans le PDF : après la page/slide 77, quand les propriétés de texte, couleurs et fonds ont été vues.

## Objectif

Créer une identité visuelle simple et lisible pour ta page.

## Énoncé

Améliore ton CSS pour donner une ambiance cohérente à ta page. Choisis une police, une couleur principale, une couleur de fond et un traitement visuel pour les zones importantes.

Le résultat doit rester confortable à lire. Les couleurs doivent aider à comprendre la page, pas gêner la lecture.

## Contraintes

- Une police générale doit être définie.
- La taille de texte générale doit être maîtrisée.
- Une couleur de texte doit être définie.
- Une couleur de fond générale doit être définie.
- Les titres doivent avoir un style différent du texte courant.
- Le `header` doit être visuellement identifiable.
- Au moins un bloc de contenu doit avoir un fond différent.
- Une couleur transparente doit être utilisée avec `rgba()`.
- Les contrastes doivent rester lisibles.

## Résultat attendu

La page doit sembler plus finie : on distingue le haut de page, les titres, les blocs et le fond général.

## Idée de rendu

<div class="render-example render-theme">
  <header>
    <h2>Nom du projet</h2>
    <p>Une phrase d'accroche courte.</p>
  </header>
  <section>
    <h3>Information importante</h3>
    <p>Un bloc clair avec une couleur de fond douce.</p>
  </section>
</div>

## Checklist

- La police générale est définie.
- Le fond de page est défini.
- Les couleurs sont cohérentes.
- Les titres ressortent.
- Le `header` est identifiable.
- Un bloc de contenu est mis en avant.
- `rgba()` est utilisé.
- Le texte reste facile à lire.

## Bonus

Ajoute une image de fond sur une zone de la page, en gardant le texte lisible.
