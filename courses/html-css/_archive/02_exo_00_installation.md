# Exercice 00 - Installation et premier dossier web

> À faire dans le PDF : après la page/slide 10, quand le navigateur, l'éditeur et les fichiers ont été présentés.

## Objectif

Préparer ton espace de travail pour tout le cours et vérifier que tu sais ouvrir une page HTML dans un navigateur.

## Énoncé

Crée un dossier principal pour le cours, puis un premier dossier d'exercice. Dans ce dossier, prépare les fichiers et sous-dossiers qui serviront pendant la formation.

Ton premier fichier HTML doit afficher une courte phrase dans le navigateur. Le but n'est pas encore d'écrire une page complète, mais de valider que ton dossier, ton fichier et ton navigateur fonctionnent correctement.

## Contraintes

- Le dossier principal doit s'appeler `cours-html-css`.
- Le dossier de l'exercice doit s'appeler `exo-00`.
- Le dossier doit contenir un fichier `index.html`.
- Le dossier doit aussi contenir un fichier `style.css`.
- Un dossier `images` doit être prévu, même s'il reste vide pour le moment.
- La page doit afficher une phrase simple dans le navigateur.

## Résultat attendu

Quand tu ouvres `index.html`, le navigateur affiche ta phrase. Tu dois pouvoir retrouver facilement ton dossier et expliquer où se trouvent les fichiers.

## Checklist

- Le dossier principal existe.
- Le dossier `exo-00` existe.
- Le fichier `index.html` existe.
- Le fichier `style.css` existe.
- Le dossier `images` existe.
- La page s'ouvre dans le navigateur.
- Le texte apparaît correctement.

## Bonus

Change la phrase affichée, recharge la page et vérifie que le navigateur montre bien la nouvelle version.
