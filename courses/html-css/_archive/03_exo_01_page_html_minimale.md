# Exercice 01 - Créer une page HTML minimale

> À faire dans le PDF : après la page/slide 18, quand la structure HTML minimale a été vue.

## Objectif

Créer une vraie page HTML structurée, avec les zones indispensables d'un document web.

## Énoncé

Dans un nouveau dossier `exo-01`, crée une page HTML complète sur un sujet simple de ton choix : profil, hobby, lieu fictif, club, association, activité, événement ou autre thème facile à présenter.

Ta page doit avoir une structure correcte, un titre dans l'onglet du navigateur et un premier contenu visible dans la page. Ajoute aussi un commentaire dans ton code pour montrer que tu sais laisser une note lisible sans l'afficher à l'écran.

## Contraintes

- Le fichier doit s'appeler `index.html`.
- La page doit déclarer le type de document.
- La langue de la page doit être indiquée.
- L'encodage doit permettre d'afficher correctement les accents.
- Le titre de l'onglet doit être personnalisé.
- Le contenu visible doit contenir au moins un grand titre et un paragraphe.
- Le code doit contenir au moins un commentaire HTML.
- L'indentation doit rendre la structure facile à lire.

## Résultat attendu

Dans le navigateur, on voit le titre principal et un paragraphe. Dans l'onglet, on voit le titre personnalisé de la page.

## Checklist

- Le fichier commence par une structure HTML complète.
- La langue est définie en français.
- Les accents s'affichent correctement.
- Le titre de l'onglet est personnalisé.
- La page contient un `h1`.
- La page contient un paragraphe.
- Un commentaire est présent dans le code.
- Le code est indenté.

## Bonus

Ajoute un deuxième paragraphe qui explique pourquoi tu as choisi ce sujet.
