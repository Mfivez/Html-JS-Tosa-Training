# Guide des exercices HTML & CSS

Ce cours sert de livret d'exercices en complément du PDF. Les apprenants ouvrent un exercice quand ils arrivent à la page ou à la slide indiquée par le formateur.

## Comment utiliser ce livret

1. Ouvrir le PDF du cours.
2. Avancer dans les notions avec le formateur.
3. Quand le PDF arrive au repère indiqué, ouvrir l'exercice correspondant.
4. Réaliser l'exercice dans son dossier de travail.
5. Vérifier le résultat avec la checklist avant de continuer.

## Progression

| Repère dans le PDF | Notions travaillées | Exercice à ouvrir |
|---|---|---|
| Après la page/slide 10 | Navigateur, éditeur, fichiers | Exercice 00 - Installation |

| Après la page/slide 27 | Titres, paragraphes, listes, mise en valeur | Exercice 02 - Texte et listes |
| Après la page/slide 37 | Liens, ancres, images, figures | Exercice 03 - Liens, images et ancres |
| Après la page/slide 47 | `header`, `nav`, `main`, `section`, `article`, `aside`, `footer` | Exercice 04 - Structure sémantique |
| Après la page/slide 56 | Liaison CSS, sélecteurs, classes, identifiants | Exercice 05 - Premiers sélecteurs CSS |
| Après la page/slide 77 | Typographie, couleurs, arrière-plans, transparence | Exercice 06 - Typographie et couleurs |
| Après la page/slide 93 | Box model, dimensions, marges, paddings, bordures, ombres | Exercice 07 - Cartes et box model |
| Après la page/slide 101 | `:hover`, `:focus`, `:active`, `:nth-child` | Exercice 08 - Pseudo-classes |
| Après la page/slide 131 | Formulaires, labels, inputs, validation | Exercice 09 - Formulaire de contact |
| Après la page/slide 144 | Tableaux, `caption`, `thead`, `tbody`, `tfoot` | Exercice 10 - Tableau d'horaires |
| Après la page/slide 151 | Viewport, media queries, responsive design | Exercice 11 - Responsive |
| Fin du cours | Récapitulatif global | Exercice 12 - Mini-site vitrine |

## Organisation du dossier apprenant

Chaque apprenant garde un dossier principal :

```text
cours-html-css/
  exo-00/
  exo-01/
  exo-02/
  ...
  exo-12/
```

Chaque exercice contient au minimum un fichier `index.html`. À partir de l'exercice 05, les apprenants travaillent aussi dans `style.css`.
